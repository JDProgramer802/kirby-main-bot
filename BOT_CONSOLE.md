# 🎮 Kirby Bot - Interactive Console Guide

## 🌟 **Control Total del Bot desde GitHub**

### **📋 ¿Qué puedes hacer?**

Ahora puedes controlar tu bot directamente desde GitHub con una interfaz interactiva:

- **📊 Ver estado** del bot en tiempo real
- **🚀 Iniciar** el bot cuando quieras
- **🔄 Reiniciar** si hay problemas
- **🔍 Probar detección** de dueño
- **📱 Enviar mensajes** de prueba

### **🎯 Cómo Usar la Consola Interactiva**

#### **1. Ve a GitHub Actions**
```
https://github.com/JDProgramer802/kirby-main-bot/actions
```

#### **2. Busca "Kirby Bot - Interactive Console"**
- Haz clic en el workflow
- Presiona el botón **"Run workflow"**

#### **3. Selecciona Comando**
- **status** - Ver estado completo del bot
- **start** - Iniciar el bot
- **stop** - Detener el bot  
- **restart** - Reiniciar el bot
- **testowner** - Probar configuración de dueño

#### **4. (Opcional) Mensaje de Prueba**
- Puedes enviar un mensaje de prueba
- El bot lo procesará y mostrará respuesta

### **🔥 Comandos Disponibles**

| Comando | Descripción | Resultado |
|---------|-------------|----------|
| **status** | Ver estado completo | 📊 Información detallada |
| **start** | Iniciar el bot | 🚀 Bot corriendo |
| **restart** | Reiniciar el bot | 🔄 Reinicio completo |
| **testowner** | Probar dueño | 🔍 Verificación de configuración |
| **stop** | Detener el bot | 🛑 Bot detenido |

### **🎮 Ejemplo de Uso**

#### **Para ver el estado:**
1. **Run workflow** → status
2. **Verás**: Estado de base de datos, dependencias, configuración

#### **Para iniciar el bot:**
1. **Run workflow** → start  
2. **Verás**: Bot iniciando y listo para WhatsApp

#### **Para probar configuración:**
1. **Run workflow** → testowner
2. **Verás**: Tu número de dueño y configuración
